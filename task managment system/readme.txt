# Task Management System

The Task Management System is a web application built using Django, a Python web framework. It provides users with the ability to manage their tasks effectively by allowing them to create, update, and delete tasks. The system includes features such as task assignment, due dates, priority levels, task filtering, and user authentication.

## Features

- User Registration: New users can register an account to access the Task Management System.
- User Authentication: Secure user authentication is implemented using Django's built-in authentication system.
- Task Creation: Users can create new tasks by providing a title, description, due date, and priority level.
- Task Update: Users can update existing tasks, modifying their title, description, due date, and priority level.
- Task Deletion: Users can delete tasks they no longer need.
- Task Filtering: Tasks can be filtered by completion status, allowing users to view completed or pending tasks.
- User-Friendly Interface: The system provides a user-friendly interface using Django's templating system to navigate and interact with tasks.

## Getting Started

1. Clone the repository: `git clone https://github.com/your-username/task-management-system.git`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Apply database migrations: `python manage.py migrate`
4. Run the development server: `python manage.py runserver`
5. Access the application in your web browser at `http://localhost:8000`
